# Percentage Increase Calculator

Free, open-source percentage increase calculator. Calculate percent change between two values instantly.

## [Percentage Increase Calculator](https://percentageincreasecalculator.app/)

Features include:
- Percentage increase and decrease
- Percent difference calculator
- Percent of a number
- Reverse percentage calculator

## Quick Calculator

A simplified version is available at: https://wsmhj.github.io/percentage-increase-calculator/

## Formula

Percentage Increase = ((New Value - Old Value) / Old Value) x 100
