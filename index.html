The Percentage Increase Formula

Calculating a percentage increase is straightforward once you understand the core formula. The percentage change between two numbers is found by dividing the difference by the original value, then multiplying by 100.

Percentage Increase Formula

% Increase = [(New - Old) / |Old|] × 100

Example: Old = 80, New = 100
= [(100 - 80) / 80] × 100
= [20 / 80] × 100
= 25% increase

Percentage Decrease Formula

% Decrease = [(Old - New) / |Old|] × 100

Example: Old = 100, New = 75
= [(100 - 75) / 100] × 100
= 25% decrease

If the result is positive, the value increased. If it is negative, the value decreased. Our calculator automatically identifies the direction for you.

Frequently Asked Questions

Common questions about calculating percentage increases and changes.

What is the difference between percentage increase and percentage difference? ▼
Percentage increase measures change from an original value to a new value, always using the original as the base. Percentage difference compares two values without designating one as the original — it uses the average of both values as the base. Use percentage increase when you have a clear "before" and "after" scenario.
Can the percentage increase be more than 100%? ▼
Yes! If a value doubles, that is a 100% increase. If it triples, that is a 200% increase. There is no upper limit to percentage increase. For example, if sales go from $10,000 to $30,000, that is a 200% increase.
How do I calculate percentage increase without a calculator? ▼
Subtract the original value from the new value to find the difference. Then divide that difference by the original value. Finally, multiply by 100 to convert to a percentage. For example: (New - Old) ÷ Old × 100.
What does a negative percentage increase mean? ▼
A negative result means the value decreased — it is actually a percentage decrease. For example, if you calculate a percentage increase and get -15%, it means the new value is 15% lower than the original value.
How is percentage increase used in business? ▼
Percentage increase is used extensively in business for tracking revenue growth, comparing year-over-year performance, calculating price changes, measuring employee salary increases, analyzing website traffic trends, and reporting financial results to stakeholders.
What's the difference between percentage increase and percentage points? ▼
These are often confused. If an interest rate goes from 2% to 3%, that is an increase of 1 percentage point, but it is a 50% increase in the rate itself. Percentage points measure the absolute arithmetic difference, while percentage increase measures the relative change from the original.
